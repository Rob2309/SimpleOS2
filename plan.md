
## Bootloader
- Loads config file
- loads modules specified in config file
- links them together
- Fills kernel header with required infos (memory map, ...)

## Kernel
- Basic terminal
- printf
- Memory management
- GDT
- Interrupts, APIC
- User mode
- Process management
- ACPI
- PCI, PCIe
- SATA
- Multi core support
